## 개발 환경 세팅

### 1. 프로젝트 Clone

1. STS 상단 메뉴 → `Window → Show View → Other → Git → Git Repositories`
2. Git Repositories 뷰 빈 공간 우클릭 → `Clone a Git Repository`
3. URI: `https://github.com/acorn1066/ddeonabom.git` 입력 → Next
4. Branch 기본값 → Next → Local destination 확인 → Finish

### 2. STS에 Import

1. `File → Import → Gradle → Existing Gradle Project` → Next
2. Project root directory: clone된 `ddeonabom` 폴더 선택 → Finish

> ⚠ 처음 import 후 Gradle 라이브러리 다운로드로 몇 분 걸립니다.
> ⚠️ 빨간 X 뜨면 프로젝트 우클릭 → `Gradle → Refresh Gradle Project`
